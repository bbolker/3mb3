Computer stuff

  • You will need to install Python/Anaconda/Jupyter Notebook.
    1. Download the 64-bit Python 3.6 installer from the anaconda installation page (click on the icon corresponding to your operating system); use it to install Python+Jupyter notebook.
    2. open a command window ("Terminal" on MacOS, "Command Prompt" or type cmd in the Start Menu in Windows) and type jupyter notebook.
